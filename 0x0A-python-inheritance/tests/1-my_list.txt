Sorted Print List Test module

==============================
Using ``MyList``
----------------

First import ``MyList`` from the ``1-my_list`` module:
	>>> MyList = __import__('1-my_list').MyList

Usage:
	>>> a = MyList()
	>>> a
	[]

	>>> a.print_sorted()
	[]

	>>> a.append(1)
	>>> a
	[1]

	>>> a.append(9)
	>>> a.append(8)
	>>> a
	[1, 9, 8]

	>>> a.print_sorted()
	[1, 8, 9]

	>>> a.append(-5)
	>>> a
	[1, 8, 9, -5]
	>>> a.print_sorted()
	[-5, 1, 8, 9]
